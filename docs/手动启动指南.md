# 手动启动指南

## 方法一：使用PowerShell脚本（推荐）

1. 右键点击 `启动服务.ps1`
2. 选择 "使用PowerShell运行"
3. 等待服务启动

## 方法二：手动命令行启动

### 步骤1：打开命令行

在 `C:\Users\youda\Desktop\new` 目录下，按住Shift键，右键点击空白处，选择"在此处打开PowerShell窗口"或"在此处打开命令提示符"

### 步骤2：进入backend目录

```bash
cd backend
```

### 步骤3：安装依赖（首次运行）

```bash
npm install
```

### 步骤4：生成Prisma客户端

```bash
npm run prisma:generate
```

### 步骤5：初始化数据库（首次运行）

```bash
npx prisma migrate dev --name init
npm run prisma:seed
```

### 步骤6：启动服务器

```bash
npm run start:dev
```

## 常见问题

### 问题1：npm不是内部或外部命令
**解决**：先安装Node.js - https://nodejs.org/

### 问题2：prisma命令找不到
**解决**：运行 `npm install` 或 `npm run prisma:generate`

### 问题3：端口3000被占用
**解决**：关闭占用3000端口的程序，或修改 backend/.env 中的PORT

### 问题4：数据库迁移失败
**解决**：删除 backend/data 文件夹，重新运行迁移命令

## 验证启动成功

看到以下提示表示启动成功：

```
✓ 数据库连接成功
🚀 服务器运行在: http://localhost:3000
📚 API文档地址: http://localhost:3000/api-docs
```
