# TabBar 图标说明

此目录用于存放 tabBar 导航图标。

## ✅ 图标已生成

扫码图标已成功生成！
- `scan.png` - 未选中状态（灰色 #7A7E83）
- `scan-active.png` - 选中状态（医疗蓝 #0ea5e9）

**尺寸**：81px × 81px
**格式**：PNG（透明背景）

## 当前配置

TabBar 有3个按钮：
1. **工作台** - 只显示文字（无图标）
2. **扫码** - 显示文字 + 图标 ⬅️ 唯一带图标的按钮
3. **患者列表** - 只显示文字（无图标）

## 重新生成图标

如果需要重新生成图标，运行以下命令：

```bash
# 方法1：使用 Python 脚本（推荐）
python generate-icons.py

# 方法2：使用 HTML 工具
# 在浏览器中打开 generate-icons.html，点击下载按钮
```

**Python 脚本需要安装 Pillow：**
```bash
pip install Pillow
```

## 图标设计

图标设计为简洁的 QR 码扫描器样式：
- 四个角落的定位框
- 中间十字形扫描线
- 线条粗细：2px
- 内边距：20%

这符合医疗应用的专业、简洁风格。
