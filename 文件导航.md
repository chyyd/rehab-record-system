## ⚡ 常用操作

### 启动后端服务
```bash
# 方式一：双击运行
test-api.bat

# 方式二：命令行
cd backend
npm run start:dev
```

### 访问API文档
```
http://localhost:3000/api-docs
```

### 查看演示
```
双击打开: demo.html
```

### 重置数据库
```bash
cd backend
rm data/database.db
npx prisma migrate dev --name init
npm run prisma:seed
```
